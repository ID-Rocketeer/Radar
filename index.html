1. RANGE RINGS

The scope displays 6 concentric green dashed range rings centered on the home station. The ring distances are scaled to the current range:

  • Ring 1 (10%): -- NM
  • Ring 2 (20%): -- NM
  • Ring 3 (40%): -- NM
  • Ring 4 (60%): -- NM
  • Ring 5 (80%): -- NM
  • Ring 6 (100%): -- NM (Scope boundary)

2. URL PARAMETERS

Modify query parameters in the address bar to configure the station location and radar range:

?lat=[latitude]&lon=[longitude]&rng=[range_nm]
  • Latitude options: lat, latitude (e.g. 30.1945)
  • Longitude options: lon, lng, long, longitude (e.g. -97.6699)
  • Range options: rng, range (clamped 2 to 250 NM)

Presets (Click to load):

  • Austin-Bergstrom (AUS) // 25 NM
  • Reagan Washington (DCA) // 10 NM
  • Nellis AFB (LSV) // 40 NM

3. SYSTEM FEATURES

  • CONSOLE STATUS: The status bezel indicates the application state. On load, the system starts in STANDBY (unlit). Once a connection is successfully established with the telemetry API, it switches to SCANNING (illuminated green). Connection loss triggers LINK ERROR (unlit).
  • CONFIGURATION TOGGLES: The configuration panel features three toggle controls whose states are indicated by illumination (illuminated green/amber for ON, dark for OFF):
    • TRAILS: Toggles historical breadcrumb tracking on the scope.
    • LOW: Filters out all high-altitude cruising traffic above the Transition Altitude of 18,000 feet (FL180), displaying only local air traffic.
    • FULLSCREEN: Expands the radar display area to fill the screen bounds.
    • CLASS B: Toggles visibility of Class B targets (gliders, balloons, UAVs) highlighted on the scope in distinct yellow.
    • SOUND TOGGLE (🔇/🔊): Located in the sidebar header. Toggles a simulated 3D room soundscape generating ambient mains hum, cooling fans, computer memory whir, air conditioning, and overhead ballast rattle echoing inside a concrete bunker room.
  • TARGET FILTERS: Restricts radar sweeps to specific target categories: ALL traffic, MILITARY ONLY, COMMERCIAL airliners, or GENERAL & OTHER aviation.
  • SELECT LOCATION: Recalibrates the home station coordinates. Click to enter calibration mode (inputs unlocked, map draggable). You can type new values directly, drag/zoom the map to reposition, or click LOCATE ME to auto-locate using your device's GPS, cell-triangulation, or IP address. Click CONFIRM to apply the new coordinates or CANCEL to abort.
